Widżet REDlink umożliwia szybkie zapisywanie użytkowników na dowolną listę adresową REDlink.

== Description ==

Wtyczka umożliwia wyświetlanie formularza zapisu do wybranej grupy kontaktów w systemie REDlink. Możesz umieścić kilka formularzy zapisu na różne grupy jednocześnie. Poza adresem e-mail (pole obowiązkowe), formularz może zbierać imię, nazwisko, nazwę firmy oraz telefon komórkowy użytkownika (pola opcjonalne).

Wtycznka używa technologii AJAX, ale działa także przy wyłączonej obsłudze JavaScript w przeglądarce użytkownika. Po udanym zapisaniu kontaktu w systemie REDlink wtyczka ustawia ciasteczko, aby użytkownik nie widział ponownie formularza zapisu na tą samą listę.

== Installation ==

1. Wgraj zawartość wtyczki (katalog redlink-widget) do katalogu /wp-content/plugins/.
1. Aktywuj wtyczkę w sekcji "Wtyczki".
1. Wprowadź login i hasło do API systemu REDlink ("Ustawienia" >> "REDlink Widget").
1. Przeciągnij widżet na swój pasek boczny w sekcji "Wygląd" >> "Widgety".
1. Skonfiguruj zachowanie widżeta.

Wymagania:

* PHP w wersji od 5.1.2
* zainstalowane rozszerzenie SOAP
